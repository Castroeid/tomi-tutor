# Tomi Tutor

אפליקציית תרגול מקורית לקריאה ולכתיבה בעברית לילדי כיתה א׳.

## מבנה נכסים (Assets)

הוספנו תיקיות ייעודיות להעלאת תמונות מקוריות:

- `assets/leo/` — העלאת אווטאר של לֵיאוֹ. אם תעלו קובץ בשם `leo-avatar.png`, הוא יוצג אוטומטית.
- `assets/banners/` — באנרים, רקעים וכותרות מקוריות.
- `assets/exercises/` — איורים לתרגילים.

> חשוב: להשתמש רק בחומרים מקוריים או ברישיון מתאים. אין להשתמש בדמויות מוגנות בזכויות יוצרים (כמו פוקימון או Brawl Stars).

## תוכן פדגוגי

כל התרגילים באפליקציה מקוריים ונוצרו בהשראת מיומנויות קריאה וכתיבה לכיתה א׳.
אין העתקה של דפי עבודה מספרים מוגנים.

## נתונים מקומיים

האפליקציה שומרת ב-`localStorage`:

- תרגילים שהושלמו
- תרגילים מאתגרים
- מצב רוח אחרון
- מטא-נתונים של הקלטות קול

## חיבור AI בעתיד
